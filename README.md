# Hō-Miâ（號名）姓名筆畫五行解析

![](https://img.shields.io/badge/Vibe%20Coding-98%25-purple?style=flat)

號名是一個姓名筆畫解析工具，根據使用者輸入的姓名，依照《周易》文字筆劃與五行理論，分析姓名中的五行屬性。同時結合三才五格原理，運用八十一數理吉凶表，為使用者解析姓名的吉凶與整體運勢。

Hō-Miâ is a lightweight web app for Chinese name numerology (stroke counts, Five Grids, Three Talents), showing element mappings and auspiciousness.

![Screenshot](./screenshot.png)
